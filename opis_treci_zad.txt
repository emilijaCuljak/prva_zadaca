Kreirala sam račun na GitHub-u prema uputama. 
Stvorila sam SSH ključ radi jednostavnosti.
Kreirala sam novi repozitorij. Koristila sam git init naredbu za to.
Koristila sam pull naredbu da bi “povukla” taj prazan repozitorij.
Pomoću naredbe git status dobila sam ispis stanja tog repozitorija.
Naredbom git add sam dodala datoteke u taj repozitorij. 
Pomoću git commit i git push sam predala repozitorij.
